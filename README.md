# command-calendar
## Installation
``` 
npm install -g @yeyunjianshi/command-calendar
```

## Usage
```
command-calendar
```
| Key |  |
|:-:| :----: |
| q | month - 1 |
| e | month + 1 |
| w | day - 7 |
| s | day + 7 |
| a | day - 1 |
| d | day + 1 |
